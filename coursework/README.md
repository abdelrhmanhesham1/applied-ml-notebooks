# Coursework & Practice

Learning-stage notebooks — Python fundamentals and core ML algorithm walkthroughs worked through while building up to the projects in [`../projects/`](../projects/).

**These are practice exercises, not showcase projects.** They're kept here (rather than hidden) because they show real fundamentals work, but they're organized separately from `projects/` so it's clear which is which — nobody should open `01_python_fundamentals.ipynb` expecting a polished case study, and nobody should read `projects/diamond-price-prediction/` as a class exercise.

## Contents

| Notebook | Topic |
|---|---|
| [`01_python_fundamentals.ipynb`](01_python_fundamentals.ipynb) | For/while loops, comprehensions, control-flow exercises |
| [`02_linear_regression_methods.ipynb`](02_linear_regression_methods.ipynb) | Simple, multiple, and polynomial linear regression |
| [`03_classification_and_titanic.ipynb`](03_classification_and_titanic.ipynb) | Logistic regression, classification metrics (confusion matrix, ROC/AUC), Titanic dataset |
| [`04_practice_exercises.ipynb`](04_practice_exercises.ipynb) | Short Python exercises — number theory, pattern printing, flow control |

## How to Run

Standard Jupyter/Colab notebooks — open any file directly (each has a Colab badge) or run locally with:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
jupyter notebook
```
